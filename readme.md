# EPERPUS — Sistem Perpustakaan Digital  
Pengembangan Sistem • 2025  
By: Putri Marshanda & Abdul Fikri

EPERPUS adalah sistem perpustakaan digital yang dirancang untuk mempermudah proses peminjaman, pengembalian, pengelolaan buku, dan administrasi perpustakaan secara efisien.  
Repository ini menyajikan pengembangan fitur baru berdasarkan proposal resmi pengembangan sistem tahun 2025.

---
##  Fitur Utama
- Login multi–role (Admin & Anggota)
- Dashboard statistik peminjaman
- Manajemen Buku (CRUD)
- Manajemen Anggota
- Peminjaman & Pengembalian Buku
- Sistem denda otomatis
- Pencarian buku berdasarkan judul/ISBN/penulis
